Welcome to WIZio-RTU home page!

This web interface provides a simple way to monitor and configure the WIZ-RTU board directly from your browser. You can use it to manage the board without writing extra scripts or sending commands manually through the REPL.

In this web interface, you can:

1. Configure the network settings, including DHCP and static IP mode,
2. Control digital relay outputs and check digital input states,
3. Read analog input values and set analog output voltages,
4. Send and receive UART messages for serial communication, and
5. Access EEPROM storage features for reading and writing saved data.

 Visit the Documentations page for more details.
