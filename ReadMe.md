# WIZio-RTU HTTP Configuration Server
Welcome to the HTTP Configuration Server.

To start your journey, upload all project files to the device.

Once the upload is complete, reset the device to start the HTTP configuration program.

If you found any Syntax Error when uploading these files, try reuploading given file and reset your device.
`main.py` should automatically run afterwards.

Visit [MicroPython for W55MH32](https://docs.w5500.com) for detailed guide.
